# الوصف التقني لتطبيق Game City 🛠️

يحتوي هذا الملف على التفاصيل التقنية الدقيقة للتطبيق، بما في ذلك جميع النقاط النهائية (Endpoints) المستخدمة في التواصل مع الخادم، بالإضافة إلى الهوية البصرية (Brand Identity) والألوان المستخدمة في التطبيق.

## 🔗 النقاط النهائية (API Endpoints)
يعتمد التطبيق على الخادم الأساسي (Base URL):
`https://orca-app-l477g.ondigitalocean.app`

### 1. المصادقة والمستخدمين (Auth & Users)
- **تسجيل الدخول:** `POST /api/users/login`
- **إنشاء حساب:** `POST /api/users/register`
- **تسجيل الدخول باستخدام جوجل:** `POST /api/users/google-login`
- **الملف الشخصي للمستخدم:** `GET /api/users/me/profile`
- **تحديث بيانات المستخدم:** `PUT /api/users/`

### 2. الألعاب (Games)
- **قائمة الألعاب:** `GET /api/games`
- **البحث أو الطلب:** `GET /api/games/search-or-request`

### 3. الأصدقاء (Friends)
- **قائمة الأصدقاء:** `GET /api/friends/list`
- **مقترحات الأصدقاء:** `GET /api/friends/suggestions`
- **إرسال طلب صداقة:** `POST /api/friends/request`
- **قبول طلب صداقة:** `POST /api/friends/accept`
- **طلبات الصداقة المعلقة:** `GET /api/friends/requests/pending`

### 4. الدردشة والمجتمع (Chat)
- **غرف الدردشة:** `GET /api/chat/rooms`
- **إرسال رسالة:** `POST /api/chat/send`

### 5. وسائل التواصل الاجتماعي (Social Media)
- **عرض وسائل التواصل:** `GET /api/socialMedia`
- **ربط وسائل التواصل:** `POST /api/socialMedia/link`

### 6. الإشعارات والأخبار وغيرها
- **الأخبار:** `GET /api/news/`
- **الإشعارات:** `GET /api/notifications`
- **أنواع بيانات المستخدم:** `GET /api/user-info/types`
- **معلومات المستخدم:** `GET /api/user-info`
- **الرئيسية (Dashboard):** `GET /api/reports/home`

### 7. القائمة المفضلة (Wishlist)
- **عرض المفضلة:** `GET /api/wishlist/my`
- **إضافة/إزالة من المفضلة:** `POST /api/wishlist/toggle`

## 🎨 الهوية البصرية والألوان (Brand Identity & Colors)
تم تصميم التطبيق ليدعم الوضعين النهاري والليلي مع هوية بصرية متميزة تعتمد على الألوان وتخصيص الخطوط لكل وضع.

### ☀️ الوضع النهاري (Light Mode - Basic Purple Theme)
- **اللون الأساسي (Primary - 60%):** `#23113E` (بنفسجي داكن جداً) - يُستخدم للأيقونات التفاعلية، الأسماء، الإعدادات والأشرطة.
- **اللون الثانوي (Secondary - 30%):** `#6D3A96` (بنفسجي مشرق) - يُستخدم للأزرار، فقاعات الدردشة، والشارات.
- **اللون المحايد (Neutral - 10%):** `#151515` (أسود تقريباً) - يُستخدم للنصوص الصغيرة وخلفيات النصوص المستقلة.
- **الخلفية (Background):** `#F8F9FA` (رمادي فاتح) لتوفير تباين قوي.
- **الأسطح والبطاقات (Surface/Card):** `أبيض (White)`.
- **الخط المستخدم:** `Alexandria` من Google Fonts.

### 🌙 الوضع الليلي (Dark Mode - Basic Gold Theme)
- **اللون الأساسي (Primary - 60%):** `#CCAF2C` أو `RGB(204, 175, 44)` (ذهبي) - يُعطي طابع التميز والألعاب الفاخر ويتناسب بشكل مثالي مع الخلفيات الداكنة.
- **اللون الثانوي (Secondary - 30%):** `#370262` أو `RGB(55, 2, 98)` (بنفسجي ليلي/مظلم) - يُستخدم للأزرار الأساسية وشارات الخصومات.
- **اللون المحايد (Neutral - 10%):** `#EEE2D2` أو `RGB(238, 226, 210)` (أبيض مائل للبيج) - نصوص التفاصيل المتفرقة والتاريخ.
- **الخلفية (Background):** `#140926` أو `RGB(20, 9, 38)` (أسود بلمسة بنفسجية عميقة).
- **الأسطح والبطاقات (Surface/Card):** `#4E008C` أو `RGB(78, 0, 140)` (لون بنفسجي متوسط).
- **الخط المستخدم:** `Cairo` من Google Fonts.

### 🛑 ألوان الحالات (Functional Colors)
- **الخطأ/التنبيه (Error):** `#D32F2F` (أحمر)
- **النجاح (Success):** `#388E3C` (أخضر)
